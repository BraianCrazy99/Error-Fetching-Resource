# 🏨 Sistema de Reservas em JavaScript

Projeto simples simulando um sistema de reservas de hotel (PMS).

## 🚀 Funcionalidades

- Cadastro de reservas
- Cálculo automático de valores
- Aplicação de descontos (PIX, cartão, etc.)
- Exibição formatada das reservas
- Controle de status (confirmada, pendente, cancelada)

## 💡 Exemplo de saída

👤 Hóspede: João da Silva  
🏨 Categoria: Premium  
🌙 Noites: 3  
💰 Total: R$ 1567.5  
📌 Status: confirmada  

---

## 🛠 Tecnologias utilizadas

- JavaScript

## 📚 Objetivo

Este projeto foi desenvolvido como prática de lógica de programação com JavaScript.
